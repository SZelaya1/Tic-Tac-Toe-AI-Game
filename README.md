# Tic-Tac-Toe AI with Pygame

---

## Features

- Play as **X** or **O**
- AI always plays **optimally**
- **Alpha-beta pruning** makes the AI faster
- Detects **wins, losses, and ties**
- Replay functionality to start a new game

---

## Requirements

- Python 3.12 or higher
- [Pygame](https://www.pygame.org/news) library
